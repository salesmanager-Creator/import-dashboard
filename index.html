🔗 Connect Your Google Sheet — One-Time Setup

You need to create a free Google Apps Script proxy (takes ~3 minutes). This solves the browser security restriction that blocks direct sheet access.

1

Open Google Apps Script

Go to script.google.com → click New Project → delete any existing code in the editor.

2

Paste this exact code

Copy the entire block below and paste it into the script editor:

const SHEET_ID = '1tFbbAf8s9qsDq4pKTDcRnCuF7mOOTVWaZim727zLNwE'; const SHEET_NAME = 'Sheet1'; function doGet(e) { const sheet = SpreadsheetApp.openById(SHEET_ID).getSheetByName(SHEET_NAME); const data = sheet.getDataRange().getValues(); const headers = data[0]; const rows = []; for (let i = 1; i < data.length; i++) { const row = data[i]; if (!row[0] || !row[1]) continue; rows.push({ date: String(row[0]).trim(), product: String(row[1]).trim(), buyer: String(row[2]).trim(), buyerCountry: String(row[3]).trim(), sellerCountry: String(row[4]).trim(), seller: String(row[5]).trim(), uom: String(row[6]).trim(), qty: parseFloat(String(row[7]).replace(/[^0-9.]/g,'')) || 0 }); } const out = ContentService.createTextOutput(JSON.stringify({ok:true,rows:rows,ts:new Date().toISOString()})); out.setMimeType(ContentService.MimeType.JSON); return out; }
3

Deploy as Web App

Click Deploy → New deployment → type = Web app → set "Who has access" to Anyone → click Deploy → copy the URL shown.

4

Paste your Web App URL below

It looks like: https://script.google.com/macros/s/AKfycb.../exec
